# Shopizer

## Prerequisites
- Colima with Kubernetes
- kubectl
- GitHub token with `read:packages` scope

## Run

```bash
./deploy.sh
```

## Access

| App | URL |
|-----|-----|
| API / Swagger | http://localhost:8080/swagger-ui.html |
| Shop | http://localhost:30200 |
| Admin | http://localhost:30300 |

Login: `admin@shopizer.com` / `password`
